# Sistema Bancario - Aula de Desenvolvimento de Sistemas 2
A interface gráfica do sistema foi desenvolvida utilizando JPanel, representando meu primeiro contato com a criação de interfaces gráficas em Java.

O projeto teve como principal objetivo praticar, na aplicação real, os conceitos aprendidos em sala de aula, como:

Programação Orientada a Objetos (POO)

Organização de classes

Interação entre interface gráfica e lógica do sistema

Este sistema foi desenvolvido exclusivamente com fins educacionais, servindo como uma etapa importante no meu processo de aprendizado e evolução na linguagem Java.
